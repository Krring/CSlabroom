## ไฟล์ในโปรเจกต์

| ไฟล์ | หน้าที่ |
|------|---------|
| `index.html` | หน้า Login + ประกาศ + สมัครสมาชิก |
| `register.html` | สมัครสมาชิก 2 ขั้นตอน |

| `admin.html` | หน้าอาจารย์ Admin |
| `style.css` | Shared CSS |
| `shared.js` | Shared data layer (localStorage) |
